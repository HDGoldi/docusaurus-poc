Plugins extend the capabilities of the 1NCE platform with services provided by 3rd party vendors. You can enable additional functionality by installing a plugin.

<Image alt="Plugin System in 1NCE.com portal" align="center" src="/img/1nce-os/1nce-os-plugins/66070e168d777e62e00778904faa1020981a9600a996c9d8ba534fecfceeef1e-b46aee7e56b2183e2b002a5c680f12e2e08f79b6a6bcaaaf638a84d23815bbb7-plugins_overview.jpg">
  Plugin System in 1NCE.com portal
</Image>

Available plugins:

* **Datacake** - Device and Network data visualization in pre-made or custom dashboards.
* **Mender** - Firmware Over-the-Air Management.
* **Tartabit** - IoT Bridge with Azure, AWS, and GCP.
* **Memfault** - Device Observability with Fault Diagnostics and Log management.
