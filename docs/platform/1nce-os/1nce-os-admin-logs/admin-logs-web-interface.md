Open the Admin Logs in [1NCE OS](https://portal.1nce.com/portal/customer/connectivitysuite) to see the latest logs from the devices. At the top, the filter can be used for searching on ICCID, Category or the Time Period. The last 5 Admin Log Events and Errors are shown on the dashboard.
