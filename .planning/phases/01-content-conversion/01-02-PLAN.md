  <action>
    1. Create `scripts/03-extract-base64.ts` (per D-05, D-06, D-07, CONV-05):

       **Input:** `docs/` directory (post-copy, post-frontmatter)
       **Output:** Extracted image files in `static/img/`, updated .md files with image references

       The script must:
       a. Find all .md files in `docs/` using `readMarkdownFiles()`
       b. For each file, scan for base64 data URIs matching the regex:
          `data:image\/(png|jpeg|jpg|gif|svg\+xml);base64,([A-Za-z0-9+/=\s]+)`
       c. For each match:
          - Determine file extension from the MIME type (png, jpeg->jpg, gif, svg+xml->svg)
          - Strip all whitespace from the base64 string
          - Decode with `Buffer.from(cleaned, 'base64')`
          - Compute output path: `static/img/{doc-relative-path}/{index}.{ext}`
            where doc-relative-path mirrors the doc's path in docs/ (per D-06)
            Example: `docs/1nce-os/device-controller.md` -> `static/img/1nce-os/device-controller/001.png`
          - Write the binary file
          - Replace the entire `data:image/...;base64,...` string in the .md file with `/img/{relative-path}`
       d. Log: total base64 images found, extracted, any failures (0-byte files)
       e. Verify each extracted file is > 0 bytes (pitfall 3 from research)

    2. Create `scripts/04-download-images.ts` (per D-05, D-06, CONV-04):

       **Input:** `docs/` directory
       **Output:** Downloaded image files in `static/img/`, updated .md files with local paths

       The script must:
       a. Find all .md files in `docs/`
       b. For each file, scan for URLs matching `https://files.readme.io/[^\s)"]+`
       c. For each unique URL:
          - Extract the filename from the URL (last path segment)
          - Compute output path: `static/img/{doc-relative-path}/{filename}`
            where doc-relative-path mirrors the doc's location (per D-06)
          - Download using `node:https` with:
            - 3 retry attempts with 1s, 2s, 4s backoff (pitfall 4 from research)
            - Follow redirects (up to 3)
            - Timeout of 10 seconds per request
          - Write the binary file to static/img/
          - Replace the URL in the .md file content with `/img/{relative-path}`
       d. Log: total URLs found, downloaded successfully, failed (with URL list)
       e. For permanently failed downloads, log a warning but do not crash the pipeline

    3. Add npm scripts:
       ```json
       "convert:base64": "tsx scripts/03-extract-base64.ts",
       "convert:download": "tsx scripts/04-download-images.ts"
       ```

    4. Run both scripts:
       ```bash
       npm run convert:base64
       npm run convert:download
       ```

  <action>
    1. Create `scripts/05-convert-images.ts` (per CONV-03, research Example 1):

       **Input:** `docs/` directory (post-base64-extraction, post-download)
       **Output:** Same files with `<Image>` JSX tags replaced by Markdown `![alt](src)` syntax

       The script must:
       a. Find all .md files in `docs/`
       b. For each file, find all `<Image ... />` JSX tags using regex:
          `<Image\s+[^>]*?\/>`
          Handle both self-closing (`<Image ... />`) and potentially non-self-closing variants
       c. For each `<Image>` tag, extract attributes:
          - `src="..."` -- the image source (should already be local /img/ path from prior scripts, or a remaining remote URL)
          - `alt="..."` -- alt text (fallback to `title="..."` if no alt)
          - `width="..."` -- preserve as HTML if width is specified (use `<img>` tag instead of Markdown for width control)
          - `align="..."` -- if "center", wrap in a div with style
       d. Convert to standard Markdown:
          - Simple case (no width/align): `![alt text](/img/path.png)`
          - With width: `<img src="/img/path.png" alt="alt text" width="80%" />`
          - With center align: wrap in `<div style={{textAlign: 'center'}}>...</div>`
       e. Handle edge cases:
          - `<Image>` tags with `caption="..."` -- add caption as italicized text below: `*caption text*`
          - `border={false}` or `border={true}` -- ignore (no visual equivalent in standard Markdown)
          - Empty alt text -- use empty string `![](/img/path.png)`
       f. Log: total `<Image>` tags found, converted, any unconverted (with file path)

    2. Add npm script:
       ```json
       "convert:images": "tsx scripts/05-convert-images.ts"
       ```

    3. Run the script:
       ```bash
       npm run convert:images
       ```
