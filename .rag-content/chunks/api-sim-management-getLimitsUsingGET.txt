# GET /v1/sims/limits

Get Global Limits

Get currently configured self-set monthly limits (Data, MT-SMS, MO-SMS) for all SIMs.

## Parameters
None

## Responses
- 200: OK
- 401: Unauthorized
- 403: Forbidden
- 404: Not Found
- 422: The following SIMs are expired and cannot be modified via API. Only extensions are permitted
