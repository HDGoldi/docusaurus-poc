# GET /v1/inspect/devices/{deviceId}/history/insights

Get Device Historian Insights

Get insights of a unique SIM device for the Historian.

## Parameters
- deviceId (path, required): The unique device ID which identifies each 1NCE SIM. Can be obtained from the Get All Devices API query. For 1NCE SIMS the device ID is equal to its iccid.
- startDateTime (query, optional): Timestamp from where onwards the historic data messages should be queried.
- endDateTime (query, optional): Timestamp until when the historic data messages should be queried.
- protocol (query, optional): Filter out data for specific protocols: <ul><li>LWM2M</li><li>UDP</li><li>COAP</li></ul>
- interval (query, optional): The interval for which grouping of statistics should occur.
