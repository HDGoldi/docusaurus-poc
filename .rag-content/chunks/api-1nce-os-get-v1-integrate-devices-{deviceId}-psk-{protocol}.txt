# GET /v1/integrate/devices/{deviceId}/psk/{protocol}

Get Pre-Shared Device Key

Get the Pre-Shared Key of a specific Device.

## Parameters
- deviceId (path, required): The unique device ID which identifies each 1NCE SIM. Can be obtained from the Get All Devices API query. For 1NCE SIMS the device ID is equal to its iccid.
- protocol (path, required): Available protocols: <ul><li>LWM2M</li><li>UDP</li><li>COAP</li></ul>
