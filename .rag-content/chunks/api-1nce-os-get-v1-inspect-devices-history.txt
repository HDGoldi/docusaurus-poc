# GET /v1/inspect/devices/history

Get Historian Messages

Get a list of historic messages for devices based on the optional filter parameters.

## Parameters
- deviceId (query, optional): The unique device ID which identifies each 1NCE SIM. Can be obtained from the Get All Devices API query. For 1NCE SIMS the device ID is equal to its iccid.
- iccid (query, optional): ICCID of a SIM device, used to identifiy each 1NCE SIM.
- startDateTime (query, optional): Timestamp from where onwards the historic data messages should be queried.
- endDateTime (query, optional): Timestamp until when the historic data messages should be queried.
- pageSize (query, optional): Parameter for specifying the queried items per page.
- protocol (query, optional): Filter out data for specific protocols: <ul><li>LWM2M</li><li>UDP</li><li>COAP</li></ul>
- nextToken (query, optional): This API query uses token-based pagination. Each queried page will include a nextToken for accessing the next page for pagination. Please use the returned token in the nextToken query parameters to obtain the next page.
