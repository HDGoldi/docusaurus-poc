Each 1NCE SIM has a private IP and is connected via the Internet Breakout using Network Address Translation to the open internet. By default the connection establishment is unidirectional from the SIM device to a server/service in the internet. The 1NCE VPN Service enables 1NCE customers to connect and transmit data bidirectional with their SIM devices via a Virtual Private Network (VPN) connection.

This section covers the setup of the 1NCE VPN client for Windows, Linux and Mac OS to establish a connection with the 1NCE Network Service. For custom OpenVPN installs, advanced routing or specific application setups refer to the .
