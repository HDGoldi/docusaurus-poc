# GET /v1/sims/{iccid}/status

Get SIM Status

Query the current status of a specific SIM card.

## Parameters
- iccid (path, required): The ICCID of the SIM to be queried.

## Responses
- 200: OK
- 401: Unauthorized
- 403: Forbidden
- 404: Not Found
- 422: The following SIMs are expired and cannot be modified via API. Only extensions are permitted
