# POST /v1/sims/simTransfer

Create SIM Transfer

Trigger a SIM transfer workflow - if possible - for moving SIMs from one customer to another.

## Parameters
None

## Responses
- 201: Created
- 401: Unauthorized
- 403: Forbidden
- 404: Not Found
- 422: The following SIMs are expired and cannot be modified via API. Only extensions are permitted
