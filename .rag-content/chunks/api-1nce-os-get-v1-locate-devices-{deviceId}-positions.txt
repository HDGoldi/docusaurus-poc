# GET /v1/locate/devices/{deviceId}/positions

Get Device Positions

Get positions of a specific device (maximum of last 7 days).

## Parameters
- deviceId (path, required): The unique device ID which identifies each 1NCE SIM. Can be obtained from the Get All Devices API query. For 1NCE SIMS the device ID is equal to its iccid.
- page (query, optional): Number of the requested page. Use this parameter to iterate through all items on the different pages. The total amount of pages is listed in the response body (pageAmount).
- sort (query, optional): Comma seperated list of device property keys in order. Prefix key with `-` to sort descending.
- source (query, optional): Source of the position.
- startDateTime (query, optional): 
- endDateTime (query, optional): 
- pageSize (query, optional): Parameter for specifying the queried items per page.
- mode (query, optional): Use ALL for a paginated list of all positions and SUMMARY to get the first, last and some positions inbetween.
