# LwM2M Integration with Location Service

LwM2M Server will automaticaly forward GPS data to [Device locator](/1nce-os/1nce-os-device-locator/index), if GPS data will be provided in following [OMA](https://raw.githubusercontent.com/OpenMobileAlliance/lwm2m-registry/prod/6.xml) resource addresses:

* `/6/0/0` (latitude, Float)
* `/6/0/1` (longitude, Float)
* `/6/0/5` (timestamp, Time). Not mandatory.

GPS data can be:

* Visualized in the 1NCE OS portal [Device inspector](/1nce-os/1nce-os-device-inspector/device-inspector-features-limitations) & [Device locator](/1nce-os/1nce-os-device-locator/index) tabs.
* Forwarded to [Cloud integrator](/1nce-os/1nce-os-cloud-integrator/index).
* Used via [API](/1nce-os/1nce-os-device-locator/device-locator-api).
