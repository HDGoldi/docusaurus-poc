# GET /v1/locate/positions/latest

Get Latest Devices Positions

Get latest positions of customer devices (maximum of last 7 days). Only one latest position is possible for a single device independent of source: either Celltower or GPS. This means that `source` query parameter selection can lead to no latest position returned for some devices.

## Parameters
- deviceId (query, optional): The unique device ID which identifies each 1NCE SIM. Can be obtained from the Get All Devices API query. For 1NCE SIMS the device ID is equal to its iccid.
- page (query, optional): Number of the requested page. Use this parameter to iterate through all items on the different pages. The total amount of pages is listed in the response body (pageAmount).
- sort (query, optional): Comma seperated list of device property keys in order. Prefix key with `-` to sort descending.
- source (query, optional): Source of the position.
