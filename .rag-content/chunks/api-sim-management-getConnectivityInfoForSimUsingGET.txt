# GET /v1/sims/{iccid}/connectivity_info

Get SIM Connectivity

Retrieve connectivity information and cell tower of a device with a given SIM. The API call returns valid information when the 1NCE SIM is attached to a 2G/3G network only.

## Parameters
- iccid (path, required): The iccid of the SIM in question.
- subscriber (query, optional): Flag indicating if the subscriber information should be retrieved. Default value is true.
- ussd (query, optional): Flag for retrieving the USSD connectivity info. Default value is false.

## Responses
- 200: OK
- 401: Unauthorized
- 403: Forbidden
- 404: Not Found
- 422: The following SIMs are expired and cannot be modified via API. Only extensions are permitted
