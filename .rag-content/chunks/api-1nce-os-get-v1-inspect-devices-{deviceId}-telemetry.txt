# GET /v1/inspect/devices/{deviceId}/telemetry

Get Device Telemetry

Get the current/last repored Shadow State (telemetry) Data for a specific Device.

## Parameters
- deviceId (path, required): The unique device ID which identifies each 1NCE SIM. Can be obtained from the Get All Devices API query. For 1NCE SIMS the device ID is equal to its iccid.
- protocol (query, optional): Filter out shadow state data for specific protocol: <ul><li>LWM2M</li><li>UDP</li><li>COAP</li></ul>
